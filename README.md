Установка БД:

1) Смонировать образ БД Project1\DB\rpoject1.mdb
2) SQL авторизация должна быть активна.
3) Выполнить скрипт на сервере Project1\DB\Script.txt

Перед запуском приложения(находится в папке Project1\App):
1) В конфигурационном файле ConS.ini изменить строку Server=*ваш сервер*.
2) Имена входа и пароли в файле Project1\App\Login.txt


Документация и диаграммы в папке Project1\Doc
